# Text to Speech Source

## ElevenLabs API

ElevenLabs has a public API that provides programmatic access to their text-to-speech, voice cloning, dubbing, and other audio AI features.

- **Availability:** Available on all plans, including the free tier (with usage limits based on characters generated). No additional cost beyond the standard generation fees.

- **Integration Options:**
  - HTTP / Websocket requests
  - Official Python library
  - Official Node.js library

- **Documentation:**
  - Full API reference: https://elevenlabs.io/docs/api-reference
  - Includes endpoints like text-to-speech conversion and more.

- **Get Started:**
  - Sign up for an account at https://elevenlabs.io
  - Generate an API key from your dashboard.
